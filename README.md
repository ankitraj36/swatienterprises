# 💊 Swati Enterprises – Premium Pharmacy E-Commerce Website

![Swati Enterprises Banner](images/hero/banner.png)

A modern, premium, fully responsive pharmacy e-commerce website built using **HTML5**, **CSS3**, and **Vanilla JavaScript (ES6+)**.

Designed with an Apple-inspired UI, clean healthcare aesthetics, accessibility, and production-quality frontend architecture.

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/swati-enterprises.git
```

Open the project:

```bash
cd swati-enterprises
```

Run using any local server.

Examples:

Using VS Code Live Server

```
Right Click
→ Open with Live Server
```

or

Python

```bash
python -m http.server
```
